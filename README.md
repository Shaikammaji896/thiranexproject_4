# Retail Sales Data Analysis Project

## Project Overview
This project performs data analysis on retail sales data to uncover sales trends, customer behavior, and revenue insights.

## Objectives
- Clean and preprocess data
- Analyze sales trends
- Visualize customer behavior
- Generate business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Higher product categories generated more revenue
- Customer age influenced purchases
- Sales patterns varied across customer groups
